## Algoritmos de Despacho

Esta aplicación permite simular y comparar diferentes **algoritmos de despacho de procesos** utilizados en los sistemas operativos.
El usuario puede agregar procesos con sus respectivos tiempos de llegada y ráfagas de CPU, visualizar los tiempos de espera, tiempos en el sistema y diagramas de Gantt para cada algoritmo.
